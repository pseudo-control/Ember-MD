Install Ember MD

GPU-accelerated molecular dynamics on Apple Silicon

Step 0 Open Terminal

Press Cmd + Space and type Terminal, then hit Enter.

Step 1 Install Homebrew (takes a few minutes)

Homebrew is a package manager for macOS. If you already have it, skip this step.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Click to copy

This will ask for your Mac password. You can also visit brew.sh for more info.

Step 2 Add the Ember tap (fast)

This tells Homebrew where to find Ember.

brew tap gentry-lab/ember
Click to copy
Step 3 Install Ember (takes a few minutes)

Downloads and installs the app to your Applications folder.

brew install --cask ember-md
Click to copy
Step 4 First launch — allow the app to open

Ember is not signed with an Apple Developer certificate, so macOS will show a warning the first time you open it. This is normal for open-source scientific software.

To open Ember after seeing this warning:

  1. Open System Settings → Privacy & Security
  2. Scroll down to the Security section
  3. You'll see "Ember MD was blocked from use because it is not from an identified developer"
  4. Click Open Anyway
  5. Enter your Mac password when prompted

Alternatively, you can right-click (or Control-click) the app in Finder and choose Open — this also bypasses the warning.

See Apple's full guide: Open a Mac app from an unknown developer

Video walkthrough: How to open apps from unknown developers on macOS

Why is Ember unsigned? Code signing requires a $99/year Apple Developer membership. Zach doesn't want to pay $100 a year to be an Apple developer because he is poor. The app is fully open-source — you can inspect every line of code on GitHub.

  Update to the latest version

Run this anytime to upgrade Ember to the newest release.

brew update && brew upgrade --cask ember-md
